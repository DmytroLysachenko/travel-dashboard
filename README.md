# Travels AI Admin Dashboard

This is the admin dashboard for the **Travels AI** platform — a travel intelligence application that helps users explore destinations, plan trips, and manage itineraries using AI-powered recommendations.

This dashboard allows administrators to manage destinations, users, analytics, and system settings. Built with **Vite**, **React**, **React Router**, and **TypeScript** for optimal performance and developer experience.

## 🚀 Tech Stack

- ⚡️ [Vite](https://vitejs.dev/) — Lightning-fast development and build tool
- ⚛️ [React](https://reactjs.org/) — Component-based UI framework
- 🛣️ [React Router](https://reactrouter.com/) — Declarative routing
- 🧠 [TypeScript](https://www.typescriptlang.org/) — Strong typing and improved developer tooling
- 💅 Optionally: [Tailwind CSS](https://tailwindcss.com/) or your preferred UI framework
